# Передача файлов в nfqws папку на роутерах Keenetic

## Как пользоваться

0. Предварительно на роутере должен стоять пакет openssh-sftp-server, установите его `opkg install openssh-sftp-server`
1. Запустите `.cmd` файл.
2. При первом запуске скрипт автоматически создаст папку `UPLOAD_FOLDER` рядом с собой.
3. Скопируйте нужные файлы в эту папку.
4. В меню скрипта нажмите `1` (Отправить файлы). папка по-дефолту на роутере /opt/etc/nfqws
5. Скрипт попросит ввести пароль от роутера.

> **Важно:** При вводе пароля символы не отображаются. Проверьте раскладку! И просто введите пароль и нажмите Enter.

### Структура меню

* `1` — Начать отправку файлов на роутер.
* `2` — Открыть папку `UPLOAD_FOLDER` в Проводнике.
* `3` — Обновить список файлов (если вы добавили файлы, пока скрипт был открыт).
* `0` — Выход.

## ⚠️ Решение проблем

### Ошибка: "Проверьте что роутер включен и Entware установлен..."
Если вы видите красное сообщение об ошибке соединения, проверьте следующее:
1. VPN ВЫКЛЮЧЕН !!!
2. Роутер включен и пингуется по указанному IP.
3. На роутере установлен пакет Entware (есть доступ к `/opt`).
4. На роутере запущен SSH-сервер (Dropbear или OpenSSH).
5. Обновлены библиотеки OpenSSL (команда `opkg update && opkg upgrade` в терминале роутера).

### Нужно ли вводить пароль каждый раз?
По умолчанию — да.
